## Naming Convention

```
[type]-[variant]-[color].png
```

**Types:**
- `icon` — S-mark only
- `lockup` — Icon + wordmark (horizontal)
- `lockup-tagline` — Icon + wordmark + tagline
- `wordmark` — Text only, no icon
- `stacked` — Vertical arrangement (if created)

**Variants (optional):**
- `tagline` — Includes "drinkable skin defense"

**Colors:**
- `navy` — Navy/dark navy (#041C72 / #242E43)
- `orange` — Orange gradient icon
- `orange-navy` — Orange icon + navy text
- `white` — White version (for dark backgrounds)
- `cream` — Cream version (for dark backgrounds)

## Usage Guidelines

| Context | Recommended Logo |
|---------|------------------|
| Favicon / app icon | `icon-navy.png` |
| Website header | `lockup-orange-navy.png` |
| Business cards / print | `lockup-tagline-navy.png` |
| Dark backgrounds | Use white or cream versions (to be added) |
| Social profile picture | `icon-navy.png` or icon in orange |

## Clear Space

Maintain clear space around the logo equal to the height of the "S" icon on all sides.

## Minimum Size

- **Icon:** 24px minimum
- **Lockup:** 120px width minimum
