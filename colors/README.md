# Sun Powder Color Palette

## Primary Palette
Use these colors for the majority of brand applications.

| Color | Hex | Usage |
|-------|-----|-------|
| **Orange** | `#FF4000` | Primary brand color, CTAs, buttons, highlights |
| **Amber** | `#FF9C00` | Gradients with orange, secondary accents |
| **Cream** | `#FFF4E8` | Light backgrounds, text on dark |
| **Navy** | `#041C72` | Headlines, strong text, dark accents |
| **Dark Navy** | `#242E43` | Body text, subtle dark elements |
| **Light Blue** | `#EBF1FF` | Light backgrounds, subtle fills |

### Common Combinations
- **Orange → Amber gradient**: Buttons, CTAs (see website)
- **Navy text on Cream background**: Primary readable combo
- **Cream text on Navy/Dark Navy**: Inverted sections
