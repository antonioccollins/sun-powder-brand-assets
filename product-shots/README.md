## Naming Convention

Use this format when adding new product shots:

```
[flavor]-[angle]-[background].png
```

**Examples:**
- `melon-sunrise-angled-blue.png`
- `melon-sunrise-front-neutral.png`
- `melon-sunrise-side-orange.png`
- `berry-blast-angled-blue.png` (future flavor)

### Angle options
- `front` — Straight-on, centered
- `angled` — Tilted/dynamic
- `side` — Profile view
- `top` — Bird's eye
- `lifestyle` — In-context/scene

### Background options
- `blue` — Blue gradient background
- `orange` — Orange gradient background
- `neutral` — Gray/white studio
- `transparent` — PNG with alpha (for compositing)

## Usage Notes

- These are high-res PNGs suitable for web and print
- For web, consider optimizing to WebP or compressing
- Use angled shots for hero sections, front shots for product grids
